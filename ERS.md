## Requisitos Funcionales

**RF-001**

El sistema debe permitir crear tareas (TODOs).

**RF-002**

El sistema debe permitir asignar un nivel de prioridad a cada tarea.

**RF-003**

El sistema debe permitir asignar una fecha a cada tarea.

**RF-004**

El sistema debe permitir configurar tareas como recurrentes.

**RF-005**

El sistema debe interpretar comandos en lenguaje natural para crear tareas.

**RF-006**

El sistema debe interpretar comandos en lenguaje natural para asignar prioridades.

**RF-007**

El sistema debe interpretar comandos en lenguaje natural para asignar fechas.

**RF-008**

El sistema debe interpretar comandos en lenguaje natural para configurar recurrencia.

**RF-009**

El sistema debe ocultar de la vista del usuario las tareas marcadas como completadas.

**RF-010**

El sistema debe conservar en la base de datos las tareas completadas con estado inactivo.

**RF-011**

El sistema debe clasificar las tareas en tiempo real mientras el usuario escribe.

---

## Requisitos No Funcionales

**RNF-001**

El sistema debe responder a cualquier operación en un tiempo máximo de 0.5 segundos.

**RNF-002**

El sistema debe estar disponible como aplicación de escritorio para Windows.

**RNF-003**

El sistema debe estar disponible como aplicación de escritorio para macOS.

**RNF-004**

El sistema debe estar disponible como aplicación de escritorio para Linux.

**RNF-005**

El sistema debe estar disponible como aplicación móvil para Android.

**RNF-006**

El sistema debe estar disponible como aplicación móvil para iOS.

**RNF-007**

El sistema debe estar disponible como aplicación web accesible desde navegador.

**RNF-008**

El sistema debe almacenar todos los datos en la nube.

**RNF-009**

El sistema debe permitir el acceso a los datos desde cualquier ubicación con conexión a internet.

**RNF-010**

El sistema debe sincronizar los cambios entre todos los dispositivos del usuario de forma instantánea.

**RNF-011**

El sistema debe reflejar los cambios sincronizados sin requerir recarga manual de la interfaz.

---

## Requisitos Técnicos

**RT-001**

El sistema debe integrarse con un servicio de modelo de lenguaje grande (LLM) para el procesamiento de lenguaje natural.
